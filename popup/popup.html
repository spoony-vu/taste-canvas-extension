Settings

Connect this extension to your own Taste Canvas deployment. Your data stays in your Vercel account — nothing flows through any shared server.

The API key is the value of TASTE_API_KEY in your deployment's environment variables.

Where to find it: Vercel dashboard → your taste-canvas project → Settings → Environment Variables → reveal TASTE_API_KEY.

Don't have one yet? Generate with openssl rand -hex 32, add it to Vercel as TASTE_API_KEY, redeploy, then paste it here. Full guide →
